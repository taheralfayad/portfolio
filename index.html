Orlando Bot

The Orlando bot acts as a neat Twitter replacement to the morning paper. It uses APIs to find the top headlines of the day that relate to the city, as well as the current temperatures. After collecting the information, it uses the Tweepy library to post a tweet with the current weather and three headlines, with the links to the headlines in the comments.

See it on GitHub!
